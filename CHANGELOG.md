# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-04-22

### Added
- **Group Management:** Users can now create groups and join existing ones using invitation codes. (Contributed by @konguito06 & @alexis-ramoss in #12)
- **User Balance Section:** Implementation of a new interface to track personal standing and net balances within a group. (Contributed by @konguito06 & @Clara-S-Correia in #12)
- **Debt Breakdown:** Added a detailed visual breakdown of "who owes whom" to simplify settlements. (Contributed by @konguito06 & @alexis-ramoss in #14)

### Changed
- **Expense Editing:** Added the capability to modify existing expenses to correct entry errors. (Contributed by @konguito06 & @luiscarlosguimaraes in #13)
